## Getting Started with MiniOS

If you're new here, MiniOS is a lightweight Linux distribution designed for:

- **Speed** - Boot in seconds, not minutes
- **Portability** - Run from USB without installation
- **Flexibility** - Multiple editions for different use cases
- **Simplicity** - Clean interface, powerful tools

![image](http://localhost:5173/assets/img/standard_00.jpg)

```bash
# Quick start: Download and create bootable USB
sudo dd if=minios-standard.iso of=/dev/sdX bs=4M status=progress
sync
```
